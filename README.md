# 📊 Index Replication Engine

A modular Python project that replicates a financial index using a subset of constituent stocks by minimizing tracking error.

This project demonstrates core concepts in **quantitative finance**, including portfolio optimization, index tracking, and performance evaluation.

---

## 🚀 Features

- 📈 Replicates index performance using selected stocks  
- 🧮 Optimizes portfolio weights to minimize tracking error  
- �� Computes and compares index vs portfolio returns  
- 🌐 Interactive Streamlit dashboard  
- 🧩 Modular and beginner-friendly code structure  

---

## 🧠 Key Concepts

- Index replication  
- Tracking error minimization  
- Portfolio optimization  
- Return computation
